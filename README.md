# 🎴 Carboot Scorekeeper

Tiny single-page web app to track scores for the card game **Carboot**.

- Add players, start a new set of games, log scores per round
- Stores everything in your browser's `localStorage` (nothing leaves your device)
- Stats: leaderboard, win rate, avg points, best/worst rounds, top rivalry
- Settings for low-wins / high-wins / target score / negative scores
- Import/export JSON of your data

## Run locally

Just open `index.html` in any browser. No build step.

## Deploy

Hosted on GitHub Pages at **https://bravostation.github.io/carboot-scorekeeper/**.

```sh
git push origin main
```

Pages picks it up from `main` (root).
